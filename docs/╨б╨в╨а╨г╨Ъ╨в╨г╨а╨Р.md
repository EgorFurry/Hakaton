# Структура проекта IncluBank

```
Hakaton/
│
├── .gitignore                  ← игнорируем мусор
├── README.md                   ← описание проекта
│
├── backend/                    ← Никита + Эгор
│   ├── main.py                 ← FastAPI, все эндпоинты
│   ├── config.py               ← настройки (порт, параметры АБС)
│   ├── requirements.txt        ← зависимости Python
│   ├── .env                    ← секреты (не пушится в git!)
│   └── .gitignore              ← локальный игнор для бека
│
├── frontend/                   ← Бекболат
│   ├── index.html              ← главный PWA файл
│   ├── manifest.json           ← PWA манифест
│   └── static/
│       ├── css/
│       │   └── style.css       ← если вынесет стили отдельно
│       └── js/
│           └── app.js          ← если вынесет JS отдельно
│
└── docs/                       ← документация
    ├── API_ДЛЯ_БЕКБОЛАТА.md   ← контракт API
    └── DEMO_SCRIPT.md          ← сценарий для защиты
```

## Правила работы с ветками

- `main`     — стабильная версия, сюда пушим только рабочий код
- `backend`  — Никита и Эгор работают здесь
- `frontend` — Бекболат работает здесь

## Когда что-то готово

```bash
# Сделал фичу — пушишь в свою ветку
git add .
git commit -m "feat: добавил синхронизацию АБС"
git push origin backend

# Когда всё готово к финалу — мержим в main
git checkout main
git merge backend
git push origin main
```
